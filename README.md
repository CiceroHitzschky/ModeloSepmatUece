# ModeloSepmatUece
É um modelo latex desenvolvido para atender a demanda de Resumos Expandidos para a Semana de Educação e Pesquisa em Matemática do Centro de Ciências e Tecnologias da Universidade Estadual do Ceará.
Com simples alterações é possível adaptar-lo para outros eventos da UECE/CCT, como a Semana Universitária.
